# AI Follow-Up & Lead Status Tracker 

This system tracks coaching leads and sends automated AI-generated follow-up messages.

It uses a simple Google Sheet as a CRM.

---

## Key Features

- Color-coded status
- Automated follow-up generation
- Organized communication
- Simple UI for non-technical users

---

## Status Colors

Green — New lead  
Blue — Follow-up lead  

---

## Tools Used

Google Sheets  
Zapier / Make  
AI Chat Model  

---

## Why This System Exists

Coaches often forget to follow up.

This system ensures:

✓ every lead gets a response  
✓ nobody is missed  
✓ more calls are booked
